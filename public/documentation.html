API documentation for movie app

This is backend side of the My Flix App. Application is used to show data, such as: title, genre, director, description... Users are able to create, delete and update account, add favorite movies and change them.

Action URL HTTP Method Path Variables Request body data format in JSON Response body data format in JSON
Allow new users to register /users POST / CREATE None { "name": "Joe", "topMovies":[] } { "name": "Joe", "topMovies":[] }
Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST / CREATE None None Returns message to inform movie is added to the list
Return a list of All users /users GET / READ None None Returns a list of an array of all users
Return a list of All movies to the user /movies GET / READ None None Returns an array of All movies
Return data about a single movie by title to the user /movies/:title GET / READ Title: Goodfellas None { "title": "Goodfellas", "genre": "Biography, Crime, Drama", "description": "The story of Henry Hill and his life in the mob, covering his relationship with his wife and his mob partners.", "director": "Martin Scorsese", "featuring": true }
Return data about a genre by name /movies/genres/:genre GET / READ Name: Action None "Action"
Return data about a director by name /directors/:director GET / READ Name: Christopher Nolan None "Christopher Nolan"
Allow users to update their info by username /users/:id PUT / UPDATE Id: 2 { "name": "updated name Mark", "topMovies":[] } { "name": "updated name Mark", "topMovies":[] }
Allow users to remove a movie from their list of favorites /users/:id/:title DELETE / DELETE Id: 2, title: Goodfellas None Goodfellas has been removed from user 2's top movies
Allow users to delete the account /users/:id DELETE / DELETE id: 1 None User {name} with id {id} is deleted.
